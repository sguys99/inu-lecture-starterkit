이 스타터킷에 대하여

INU 강의 수강생이 정적 웹서비스를 빠르게 시작하고 GitHub Pages(github.io)에 배포할 수 있도록 만든 템플릿입니다. 프레임워크나 빌드 도구 없이 HTML · CSS · JavaScript 만으로 동작하므로, 저장소에 푸시하면 그대로 웹에 공개됩니다.

구성

화면 스타일은 루트의 DESIGN.md 가 단일 출처입니다. 그 토큰을 css/tokens.css 의 CSS 변수로 옮겨 두었고, 모든 컴포넌트는 이 변수만 참조합니다. 새 색상이나 간격을 인라인으로 넣지 말고 토큰을 사용하세요.
